## Installation

```
virtualenv .env
. .env/bin/activate
pip install -r requirements.txt
```

For PostgreSQL

```
pip install -r requirements-postgres.txt
```

For MySQL

```
pip install -r requirements-mysql.txt
```

# Usage

```
./script.py --help
```
